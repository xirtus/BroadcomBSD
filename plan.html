Safe Boot Procedure

/bsd.safe is a stock GENERIC kernel WITHOUT b43bsd. If the new kernel panics or hangs, boot /bsd.safe at the bootloader prompt. This kernel has zero b43bsd symbols — it will NOT attempt to probe the Broadcom WiFi device. No more "broadcom r15" stalls.

At the boot> prompt:

boot /bsd.safe

Staged testing procedure:

# Stage 1: Boot with driver disabled to verify kernel is stable
boot /bsd -c
UKC> disable b43bsd
UKC> quit

# Stage 2: If Stage 1 passes, boot with driver enabled
boot /bsd        # normal boot

# If anything goes wrong:
boot /bsd.safe   # stock kernel, no b43bsd
